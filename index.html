About↻

WebMMU is a comprehensive benchmark designed to push the boundaries of AI for the web. It challenges models to answer questions about websites, edit real HTML/CSS/JS code, and generate web layouts from mockups‚Äîacross four languages and 20+ domains. Whether you're building smarter web agents or testing the limits of multimodal models, WebMMU is your go-to testbed.

Dataset Overview↻

WebMMU is a large-scale, multilingual benchmark for evaluating web understanding and code generation. It covers real-world websites across 20+ domains and four languages, with expert-verified annotations for three core tasks: WebQA, Mockup2Code, and Code Editing.

  • Languages: English, Spanish, German, French
  • Domains: Shopping, booking, sports, technology, and more
  • Tasks: WebQA (visual question answering), Mockup2Code (design-to-code), Code Editing (HTML/CSS/JS edits)
  • Annotations: Human-verified, high-quality, multi-step reasoning and UI understanding
  • Data: Full-page screenshots, HTML, and assets

Tasks↻

WebQA

Answer questions about real website screenshots‚Äîtest your model's ability to reason, ground, and understand UI elements and content.

Mockup2Code

Turn hand-drawn or digital mockups into working HTML/CSS code. Evaluate how well your model translates design into code.

Code Editing

Edit real website code based on user instructions. Can your model make precise, functional changes to HTML, CSS, or JS?
